# 🎮 Tic Tac Toe - Android App
A fun and interactive **Tic Tac Toe Game** built with ❤️ using **Android Studio**.  
Challenge your friends and enjoy smooth gameplay with a clean UI.

---

## ✨ Features
✅ Two-player mode  
✅ Winner highlight 🎉  
✅ Reset button ↩️  
✅ Smooth and responsive UI ⚡  

---

## 🛠️ Tech Stack
- 🖥 **Language:** Kotlin  
- 🎨 **UI:** XML  
- 🏗 **IDE:** Android Studio
- **Database:** Firebase
